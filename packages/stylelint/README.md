## Description

This package contains linting rules for SCSS files.

## Installation

```shell
npm install -D @builtbysasha/stylelint
```

## Setup

In the root of the project, create a file `.stylelintrc` with the following content:

```json
{
  "extends": "@builtbysasha/stylelint"
}
```
